# Tic Tac Toe game | Player vs Computer 
https://githubbox.com/nishesh96/tic-tac-toe-AI

Built using Vanilla Javascript, HTML, CSS 

# How to play 
Open this link -  https://githubbox.com/nishesh96/tic-tac-toe-AI
or to play locally,
1) Clone the project - 
2) Open index.html.


# Steps to Play 
1) Enter your name
2) Select X or O
3) Hit play.
4) Enjoy.
